# EOL Measurement System Analysis with ANOVA and Decision Risk

## Project overview
This project investigates the agreement between an End-of-Line (EOL) testing machine and an external parallel measurement setup using pressure and airflow data.

The analysis focuses on:
- comparison between two independent measurement sources
- ANOVA-based statistical evaluation
- probability density function (PDF) analysis
- false pass / false fail decision risk
- drift analysis over time
- reproducible research principles

## Data sources
1. EOL machine measurements
2. External parallel measurements from:
   - CEM DT-8890 pressure meter
   - TSI 5200 airflow meter
